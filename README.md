# BryanProgramImaging2026
Project by Bryan Program 2026 Cohort on classifying tetrahymena images.

This repository is organized to support our topological data analysis (TDA) workflow. The core codes are contained within our named working script files, an experiments directory features the updated and finalized implementations. Additionally, the synthetic results folder houses outputs and analyses generated by running our filtration pipelines across controlled synthetic datasets to benchmark performance and validate the topological feature extraction.
